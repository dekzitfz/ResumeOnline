Profil

Saya adalah seseorang yang sangat tertarik dan memiliki passion pada dunia aplikasi mobile, terutama pada platform android. Selama hampir tiga tahun menekuni pemrograman android, saya sudah menghasilkan beberapa aplikasi. Sebagian sudah ada di Google Play Store dan sebagian adalah project open source yang saya bagikan di Github.

Dalam proses pengembangan aplikasi android, saya menggunakan Material Design Pattern sebagai acuan tampilan antar muka dan menggunakan MVP Pattern untuk arsitektur kode pemrograman. Saat ini saya menggunakan JAVA sebagai bahasa pemrogaman native untuk memaksimalkan performa dan kebutuhan aplikasi.

Perkembangan teknologi aplikasi android sangatlah cepat, untuk itu saya juga selalu mengikuti perkembangannya melalui grup komunitas, berita, dan mailing list.
